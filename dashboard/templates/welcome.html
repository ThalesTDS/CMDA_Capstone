How We Evaluate Documentation

  • Comment Density: % of lines that are comments — helps assess whether code is well-documented.
  • Completeness: Checks if key elements like inputs/outputs are explained.
  • Accuracy: Are the comments truthful and aligned with what the code does?
  • Conciseness: Measures clarity without unnecessary repetition.
  • Overall Score: A combined score reflecting all of the above.
